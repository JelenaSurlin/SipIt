# SipIt 

Web aplikacija za pretraživanje koktela i barova razvijena kao timski projekt u sklopu kolegija Programiranje za Internet.

## Opis projekta

Sipit omogućuje korisnicima pregled i pretraživanje koktela te pronalazak barova u kojima je određeni koktel dostupan. Aplikacija koristi relacijsku bazu podataka za povezivanje koktela, sastojaka i lokacija.

## Funkcionalnosti

- pretraživanje koktela i barova
- filtriranje podataka prema različitim kriterijima
- pregled informacija o koktelima
- prikaz barova u kojima je dostupan određeni koktel
- povezivanje podataka putem relacijske baze podataka

## Tehnologije

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Cilj projekta

Cilj projekta bio je razviti funkcionalnu web aplikaciju kroz timski rad te primijeniti znanja iz razvoja web aplikacija, rada s bazama podataka i implementacije dinamičkog sadržaja.

## Autori

Timski projekt izrađen u sklopu fakultetskog kolegija Programiranje za Internet.
